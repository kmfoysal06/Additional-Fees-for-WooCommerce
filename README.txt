Let Users Setting Up Additional Fee For Their WooCommerce Shop

=== Description ==
Additional Fees for WooCommerce is A WordPress Plugin That Able Users Setting Up Additional Fee For Their WooCommerce Shop. Users Can Also Add Additional Fees For Payment Methods.

=== Installation ===
1. Upload the `additional-fees-wc` Folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
2. Visit WooCommerce->Settings from Admin Dashboard and Set Up Additional Fees on Extra Fee Tab.
